# Form Settings Plugin

WordPress plugin for managing Contact Form 7 recipients, validation rules, email templates, and error logging.

## Features

- **Recipients Management**: Automatically manage email recipients for Contact Form 7 forms
- **Validation Rules**: Set custom validation rules for form fields
- **Form Scanner**: Scan forms to find email fields and add them to recipients
- **Email Templates**: Create and manage custom email templates
- **Error Logging**: Track and view form submission errors
- **Copy/Paste Control**: Disable copy/paste functionality on forms
- **JS Conflict Scanner**: Detect JavaScript conflicts that may interfere with CF7

## Installation

1. Upload the `form-settings` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Form Settings' in the WordPress admin menu

## Requirements

- WordPress 5.0 or higher
- Contact Form 7 plugin installed and activated
- PHP 7.0 or higher
